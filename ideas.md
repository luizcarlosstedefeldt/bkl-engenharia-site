# Ideias de Design - BKL Engenharia

## Abordagem Escolhida: Modernismo Corporativo com Acentos Quentes

### Design Movement
**Modernismo Corporativo Elevado** - Inspirado em arquitetura contemporânea e design industrial, com ênfase em precisão, clareza e profissionalismo. Incorpora elementos de engenharia visual (linhas limpas, proporções áureas, hierarquia clara) combinados com acentos quentes que transmitem energia e confiança.

### Core Principles
1. **Clareza Estrutural**: Hierarquia visual nítida, navegação intuitiva, informação bem organizada
2. **Profissionalismo Elevado**: Tipografia sofisticada, espaçamento generoso, paleta de cores corporativa com acentos
3. **Confiança através da Solidez**: Uso estratégico de peso visual, sombras sutis, elementos que transmitem estabilidade
4. **Dinamismo Controlado**: Transições suaves, micro-interações refinadas que não distraem do conteúdo

### Color Philosophy
- **Primária**: Azul profundo (`#1e3a5f` - confiança, profissionalismo, engenharia)
- **Secundária**: Laranja quente (`#ff6b35` - energia, inovação, progresso)
- **Neutros**: Cinzas sofisticados (`#f5f5f5`, `#2c2c2c`) para fundo e texto
- **Acentos**: Branco puro para contraste, cinza claro para elementos secundários
- **Raciocínio**: Azul transmite confiança e profissionalismo (essencial para engenharia). Laranja adiciona energia e diferencia a marca, criando pontos de interesse visual sem parecer corporativo demais.

### Layout Paradigm
- **Hero Section Assimétrica**: Imagem de obra grande à direita, texto e CTA à esquerda com fundo azul profundo
- **Grid Fluido**: Seções alternando entre layouts full-width e containers centrados
- **Diagonal Dividers**: Transições entre seções usando cortes diagonais sutis (clip-path) para criar movimento
- **Sidebar Contextual**: Informações complementares em painéis laterais em seções específicas
- **Evitar**: Layouts centrados genéricos, simetria excessiva

### Signature Elements
1. **Linhas Diagonais Sutis**: Usadas em divisores de seção e backgrounds para criar movimento
2. **Cards com Sombra Elevada**: Efeito de profundidade em portfólio e depoimentos
3. **Ícones Geométricos**: Ícones customizados com formas geométricas (quadrados, triângulos) para reforçar tema de engenharia

### Interaction Philosophy
- **Hover Effects Refinados**: Elevação sutil, mudança de cor laranja em CTAs
- **Scroll Animations**: Fade-in suave para elementos, parallax sutil em hero
- **Transições Fluidas**: 300-400ms para mudanças de estado
- **Feedback Visual**: Botões mudam para laranja ao hover, links sublinham sutilmente

### Animation
- **Entrance**: Fade-in + slide-up (200ms) para cards de portfólio
- **Hover**: Elevação 4px + sombra aumentada para cards
- **Scroll**: Parallax leve (velocidade 0.5) na hero image
- **Micro-interactions**: Ícones rotacionam 5° ao hover, transição suave de cor em 150ms

### Typography System
- **Display (H1)**: Poppins Bold 48px, azul profundo, letter-spacing -0.5px
- **Heading (H2)**: Poppins SemiBold 32px, azul profundo
- **Subheading (H3)**: Poppins Medium 20px, cinza escuro
- **Body**: Inter Regular 16px, cinza escuro, line-height 1.6
- **Small**: Inter Regular 14px, cinza médio
- **Pairing**: Poppins (display, headings) + Inter (body, UI) para contraste e legibilidade

## Alternativas Descartadas

### Opção 2: Minimalismo Escandinavo (Descartada)
Seria muito genérico e não diferenciaria a marca. Falta de energia para uma empresa de engenharia.

### Opção 3: Design Brutalist (Descartada)
Muito extremo e poderia parecer frio demais para um site corporativo de engenharia.
